# Запуск тестов и работа с Allure-отчетом

## Описание
Проект содержит автотесты для проверки функционала калькулятора и интернет-магазина.

## Предварительные требования
Убедитесь, что у вас установлены:
*   Python 3.x
*   Библиотеки из `requirements.txt` (установка: `pip install -r requirements.txt`)
*   Allure Commandline

## Запуск тестов для формирования отчета
Чтобы запустить все тесты и сохранить результаты для генерации отчета, выполните команду в терминале:

    pytest --alluredir=allure-results

## Просмотрите отчет:

    allure serve allure-results

## Для автоматического запуска можно использовать файл

    run_tests.bat

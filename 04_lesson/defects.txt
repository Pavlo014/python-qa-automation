1. Краткое описание
    Метод contains(string: str, symbol: str) -> bool возвращает True при поиске пустой строки ("") в непустой строке, что противоречит ожидаемому поведению и документации.

2. Шаги для воспроизведения
    - Создать экземпляр класса: string_utils = StringUtils()
    - Вызвать метод: result = string_utils.contains("test", "")
    - Проверить результат: print(result)  # Выводит `True`, но ожидается `False`

3. Фактический результат
    Для ввода ("test", "") метод возвращает True.

4. Ожидаемый результат
    Для ввода ("test", "") метод должен возвращать False, так как:
    пустая строка не является «искомым символом» в контексте бизнес‑логики;
    это соответствует тестовому сценарию test_contains_negative[test--False].

5. Причина дефекта
    В коде метода отсутствует явная проверка на пустой символ:
    try:
        res = string.index(symbol) > -1  # `index("")` всегда возвращает 0 для непустой строки
    except ValueError:
        pass
